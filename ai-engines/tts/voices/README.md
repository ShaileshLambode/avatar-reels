# XTTS v2 Custom Voice Reference WAVs

If you want to use custom voice cloning instead of the internal default speakers, you can place 6–10 second clean, mono WAV files here.

## Reference WAV Guidelines:
1. Format: `.wav` (PCM 16-bit, 22050Hz or 24000Hz mono recommended)
2. Content: 6–10 seconds of clear speech, without background noise or music
3. Naming: Filename determines the voice ID (e.g. `my_voice.wav` -> `my_voice`)

*Note: For Phase 1D, we support both custom cloning WAV files placed in this folder and the built-in default XTTS speaker preset.*
